# DerMas-Lab

DerMas Lab es un espacio de dermocosmética y cuidado personal donde seleccionamos productos confiables de skincare y bienestar con criterio profesional, estética moderna y respaldo farmacéutico. Buscamos ofrecer una experiencia simple, premium y enfocada en el cuidado real de la piel.


## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo

Desarrollar una web moderna, minimalista y responsive que represente la identidad visual y profesional de la marca.
